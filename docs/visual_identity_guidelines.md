# 腾讯云 × 创芽OPC社区 “AI Agent助力OPC创业专场” 视觉语言规范 (VI Guidelines) - 方案A 浅色版

本规范旨在为“腾讯云 AI Agent 助力 OPC 创业专场”活动提供统一、专业且极具科技感与亲和力的视觉指导。本方案采用**方案A（浅色系 + 薰衣草紫）**，营造轻盈、现代、极具SaaS产品感的视觉调性。

---

## 1. 色彩系统 (Color Palette)

色彩系统以**纯白**和**极浅蓝紫**作为大面积底色，融合**腾讯科技蓝**展现专业度，并点缀**薰衣草浅紫**与**创芽活力绿**，展现AI科技与生机勃勃的个人创业力量。

| 颜色名称 | 十六进制 (HEX) | RGB 值 | 视觉定位与应用场景 |
| :--- | :--- | :--- | :--- |
| **纯白底色** | `#FFFFFF` | `rgb(255, 255, 255)` | **主背景色**。用于页面和海报的大面积底色，营造干净、通透、呼吸感十足的现代设计空间。 |
| **腾讯科技蓝** | `#0052D9` | `rgb(0, 82, 217)` | **主品牌色**。用于主要标题、重点按钮、腾讯云Logo，传递专业、稳重的科技感。 |
| **薰衣草浅紫** | `#C4B5FD` | `rgb(196, 181, 253)` | **AI主题色**。用于副标题、发光特效、渐变过渡及活动亮点，代表AI Agent的智能与未来感。 |
| **创芽活力绿** | `#00C35A` | `rgb(0, 195, 90)` | **社区特色色** | 用于创芽Logo、幼芽元素、重点强调标签，传递生命力与创新精神。 |
| **科技浅紫背景**| `#F5F3FF` | `rgb(245, 243, 255)` | **辅助背景色**。用于卡片背景、内容板块区隔，增加页面视觉层次。 |
| **深邃碳灰** | `#1D2129` | `rgb(29, 33, 41)` | **主文字色**。用于大段正文、说明文字，保证在浅色背景下的极高可读性。 |

### 渐变色规范 (Gradients)
*   **主渐变（科技幻彩渐变）**：`linear-gradient(135deg, #0052D9 0%, #C4B5FD 100%)`
    *   *应用*：主标题文字、大面积背景微光、主视觉渐变条。
*   **生态渐变（蓝绿科技渐变）**：`linear-gradient(135deg, #0052D9 0%, #00C35A 100%)`
    *   *应用*：创芽社区相关板块、轻量化创业（OPC）主题标签。

## 2. 字体与排版规范 (Typography)

为了在手机端（移动端）和电脑端（PC端）均获得最佳的清晰度与科技感，字体选用现代无衬线体。

### 推荐字体族 (Font Family)
*   **中文首选**：`PingFang SC`, `Microsoft YaHei`, `Alibaba PuHuiTi`, `sans-serif`
*   **英文/数字首选**：`Segoe UI`, `Helvetica`, `Arial`, `sans-serif`

### 字号与行高层级 (Typography Hierarchy)

| 层级名称 | 推荐字号 (PC / Mobile) | 行高 (Line Height) | 字重 (Font Weight) | 应用场景示例 |
| :--- | :--- | :--- | :--- | :--- |
| **超大标题 (H1)** | `36px` / `28px` | `1.3` | **Bold (700)** | 活动主标题（蓝紫渐变，建议向右倾斜 6° 增加动感） |
| **一级标题 (H2)** | `24px` / `20px` | `1.4` | **Bold (700)** | 模块标题（如：嘉宾介绍、专场议程） |
| **二级标题 (H3)** | `18px` / `16px` | `1.5` | **Medium (500)** | 嘉宾姓名、卡片标题 |
| **标准正文 (Body)** | `14px` / `14px` | `1.6` | **Regular (400)** | 嘉宾介绍、活动详情、排版正文 |
| **辅助说明 (Caption)**| `12px` / `11px` | `1.5` | **Regular (400)** | 脚注、免责声明、扫码提示文字 |

## 3. 品牌Logo应用规则

本次活动为**腾讯云**与**创芽OPC社区**联合主办，Logo呈现需遵循以下规范：

```
+-------------------------------------------------------------+
|    [ 腾讯云 Logo ] (深蓝色/浅蓝色)   |   [ 创芽OPC 社区 Logo ]      |
+-------------------------------------------------------------+
```

### 1. 组合规则
*   在海报和网页的头部，双方Logo必须并排呈现。
*   **左侧**为腾讯云Logo，**右侧**为创芽OPC社区Logo（或创芽OPC形象Logo），中间用一条灰色的竖线 `|` 隔开。
*   在浅色背景下，Logo必须使用其原始的彩色版本或深色文字版本，确保高清晰度和识别度。

### 2. 颜色适应
*   **Logo形状不能更改**。
*   若海报整体需要高纯净度色调，可将Logo的文字部分适当调整为**腾讯科技蓝**（`#0052D9`）或**深碳灰**（`#1D2129`），但图形部分（腾讯云的云朵、创芽的数字1与绿芽）需保持其原本的品牌特征。

## 4. 图形与视觉元素 (Graphics & Motifs)

### 核心视觉符号
1.  **3D 科技云与 AI 轨道**：代表腾讯云强大的云端算力与 AI Agent 生态。
2.  **创芽小仓鼠 (IP形象)**：代表“One Person Company”的轻量化创业者，头顶绿色幼芽，增添亲和力，打破纯科技的冰冷感。
3.  **磨砂玻璃质感 (Glassmorphism)**：
    *   *实现方式*：半透明白色背景（如 `rgba(255, 255, 255, 0.7)`）配合背景模糊（`backdrop-filter: blur(10px)`）。
    *   *描边*：1px 的浅蓝/浅紫半透明描边，模拟精致的玻璃边缘。
4.  **发光线条与科技微光**：在背景中使用大半径的柔和彩色投影（Drop Shadow），营造未来感。
